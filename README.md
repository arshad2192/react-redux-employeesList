# React Redux Employees List

A simple web application built with React and Redux that displays a list of employees.

## 🚀 Features

- Display employees list
- State management using Redux
- Clean and simple UI
- Component-based architecture

## 🛠️ Tech Stack

- React
- Redux
- JavaScript
- HTML/CSS

## 📦 Installation

```bash
git clone https://github.com/arshad2192/react-redux-employeesList.git
cd react-redux-employeesList
npm install
npm start
```
